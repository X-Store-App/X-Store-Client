## Building and running

### Prerequisites

- [Node.js](https://nodejs.org)
- [Yarn](https://yarnpkg.com/)

To start, clone this repo:
```
git clone https://github.com/X-Store-App/client.git
cd client
```

Install its dependencies:
```
yarn
```

And run it:
```
yarn dev
```

To package it, run:
```
yarn package
```
