# XStore Cross Platform client

## Technologies

*This project uses the XStore's Node API. Check its repo [here.](https://github.com/X-Store-App/api-node) or (prefered way) you can download it from npm/yarn:*
```bash
npm i @xstoreapp/api
# or
yarn add @xstoreapp/api
```

## Available scripts

```bash
yarn start
```

```bash
yarn package
```

```bash
yarn webpack:bundle
```
